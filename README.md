## THE TECH STACK
* **Core:** Vanilla HTML5 / CSS3 / JavaScript
* **Physics:** Solid-body collision logic with vector-based momentum.
* **Audio:** Real-time synthesis via the `Web Audio API`.
* **Hitbox:** Precision cursor-tracking with visual white-ring feedback.
